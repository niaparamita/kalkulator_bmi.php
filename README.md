# 💪 BMI Calculator Premium

BMI Calculator Premium adalah website kalkulator BMI (Body Mass Index) berbasis web yang dibuat menggunakan PHP, HTML, CSS, dan JavaScript.  
Website ini digunakan untuk menghitung indeks massa tubuh seseorang berdasarkan berat badan dan tinggi badan yang dimasukkan oleh pengguna.

Program ini menampilkan hasil BMI secara otomatis beserta kategori berat badan dan tips kesehatan dengan tampilan modern dan interaktif.

---

# 📖 Penjelasan Program

BMI (Body Mass Index) adalah metode perhitungan yang digunakan untuk mengetahui apakah berat badan seseorang tergolong ideal, kurang, berlebih, atau obesitas berdasarkan tinggi badan.

Pada website ini, pengguna cukup memasukkan:
- Nama
- Berat badan
- Tinggi badan

Kemudian sistem akan:
1. Menghitung nilai BMI
2. Menentukan kategori BMI
3. Menampilkan hasil BMI
4. Memberikan tips kesehatan sesuai kategori BMI

---

# 🧮 Rumus BMI

Rumus BMI yang digunakan pada program:

\[
BMI = \frac{Berat\ Badan}{(Tinggi\ Badan\ dalam\ meter)^2}
\]

Contoh:

- Berat badan = 55 kg
- Tinggi badan = 170 cm

\[
BMI = \frac{55}{(1.70)^2}
\]

Hasil:

\[
BMI = 19.0
\]

Kategori:
✅ Normal
